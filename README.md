# Ion_Dependent_Rate_Constants_Prediction
Machine learning tool for prediction of reaction rate constants between NO3 radicals and VOCs in the atmospheric aqueous phase.  
## Citation
The paper "Predicting Nitrate Radical Reaction Rate Constants of Organic Compounds in the Atmospheric Aqueous Phase Affected by Inorganic Ions" is published in Environmental Science & Technology as ASAP article. You can read our paper [here](https://doi.org/10.1021/acs.est.5c10685).
